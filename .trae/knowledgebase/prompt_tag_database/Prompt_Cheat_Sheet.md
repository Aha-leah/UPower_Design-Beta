# Figma Make Prompt Cheat Sheet (2025+ Edition)
<!-- 风格标签库 -->

此文档用于在生成 Brand DNA 和 Style Prompt 时提供标准化的视觉语言。AI 应优先从中选取关键词。

| Category | Keywords (English) | Visual Effect / Intention |
| :--- | :--- | :--- |
| **Styles (主流风格)** | **Post-Digital Naturalism** | 后数字自然主义。数字与有机的结合，柔和的颗粒感，仿生形态，暖色调科技。 |
| | **Spatial UI / VisionOS** | 空间计算风格。多层深度，毛玻璃，视差悬浮，眼动追踪友好的大间距。 |
| | **Hyper-Utilitarian** | 超级实用主义。高密度信息展示，类似 Bloomberg 终端，无装饰，强调数据可视化效率。 |
| | **Abstract 3D Surrealism** | 抽象 3D 超现实。使用 Spline/Blender 渲染的抽象几何体，反重力，高饱和度材质。 |
| | **Y2K Reimagined** | 新千年复古。像素艺术，亮粉/亮绿，粗糙边缘，复古浏览器窗口元素。 |
| | **Dark Academic** | 深色学术风。衬线体，米色/深棕配色，纸张质感，知识密集型氛围。 |
| **Materials (前沿材质)** | **Liquid Metal** | 液态金属。流动的水银质感，高反光，动态扭曲。 |
| | **Frosted Acrylic** | 磨砂亚克力。比传统毛玻璃更厚重，有物理厚度感，边缘高光。 |
| | **Holographic Foil** | 全息烫金。随光线变化的彩虹色泽，用于强调卡片或徽章。 |
| | **Generative Noise** | 生成式噪点。动态的颗粒背景，增加触感和温暖度。 |
| | **Vantablack** | 极黑。OLED 纯黑背景，配合极高亮的霓虹线条。 |
| **Lighting (高级光影)** | **Volumetric Lighting** | 体积光/丁达尔效应。有空气感的光束，营造空间深度。 |
| | **Caustics** | 焦散。水波纹或玻璃折射投下的复杂光斑。 |
| | **Ambient Occlusion** | 环境光遮蔽。极度真实的阴影细节，强调物体接触面。 |
| | **Bioluminescence** | 生物发光。像深海生物一样的自体发光，神秘且有机。 |
| **Layout (现代布局)** | **Bento Grid 2.0** | 进阶便当盒。更复杂的非对称拼图，支持视频/3D 互动的卡片。 |
| | **Scroll-Triggered Canvas** | 卷轴画布。整个页面是一个巨大的 Canvas，随滚动无缝变形。 |
| | **Dynamic Split** | 动态分割。分割线可拖拽，或随鼠标位置改变比例。 |
| | **Island UI** | 岛屿式布局。导航栏、工具栏悬浮在内容之上，像灵动岛一样收缩。 |
| **Typography (排版趋势)** | **Kinetic Typography** | 动态排版。文字随滚动变形、变色或跟随路径。 |
| | **Editorial Serif** | 杂志级衬线。高对比度衬线体 (如 GT Super)，用于大标题，强调阅读感。 |
| | **Variable Fonts** | 可变字体。利用字重/宽度的平滑变化来响应交互。 |

## 常用指令组合 (Combos 2025)

*   **未来生态 (Eco-Futurism):** "Post-Digital Naturalism + Generative Noise + Serif Headings + Earthy Tones + Organic Shapes"
*   **沉浸空间 (Spatial Computing):** "VisionOS Style + Frosted Acrylic + Volumetric Lighting + Floating Islands + Depth Parallax"
*   **硬核数据 (Pro-Data):** "Hyper-Utilitarian + Monospace + High Contrast + Visible Grid Lines + Real-time Data Viz"
*   **流体品牌 (Liquid Brand):** "Abstract 3D Surrealism + Liquid Metal Materials + Kinetic Typography + Caustics Lighting"
